ระบบล็อกการเชื่อมโยงข้อมูล

กรุณากรอกรหัสผ่านเพื่อเข้าสู่การตั้งค่าระบบ

รหัสผ่านไม่ถูกต้อง กรุณาลองใหม่อีกครั้ง

แผงควบคุมตั้งค่าระบบส่งข้อมูล

จัดการข้อมูล Google Sheets และระบบแจ้งเตือนผ่าน Telegram

หลังจากนำโปรเจกต์สคริปต์ไปใช้เป็นแอปพลิเคชันเว็บแล้ว ให้คัดลอก URL มาใส่ที่นี่

ขั้นตอนการตั้งค่าใช้งาน (เชื่อมไปยังชีตเป้าหมายหลัก):

  1. เปิด Google Sheet ของคุณ คลิกที่นี่เพื่อเปิดแผ่นงาน
  2. เลือกเมนู ส่วนขยาย (Extensions) > Apps Script
  3. ลบโค้ดเริ่มต้นออกให้หมด แล้วคัดลอกโค้ดสคริปต์จากด้านล่างนี้ไปวาง
  4. กดปุ่ม บันทึก (รูปแผ่นดิสก์)
  5. กดปุ่ม การทำให้ใช้งานได้ (Deploy) > การทำให้ใช้งานได้ใหม่ (New deployment)
  6. เลือกประเภทเป็น แอปพลิเคชันเว็บ (Web app)
  7. ตั้งค่า ผู้มีสิทธิ์เข้าถึง (Who has access) เป็น ทุกคน (Anyone)
  8. กดปุ่ม deploy จากนั้นคัดลอก URL แอปพลิเคชันเว็บ มาวางในช่องด้านบน

กลุ่มจะเริ่มต้นด้วยเครื่องหมายลบ เช่น -100xxxxxxxxx

วิธีการหา Chat ID ใน Telegram:

  • รับแจ้งเตือนส่วนตัว: ค้นหาบอทชื่อ @userinfobot ใน Telegram กด Start เพื่อรับ ID ของคุณมาวาง
  • รับแจ้งเตือนเข้ากลุ่ม: ดึงบอทของคุณเข้ากลุ่มปฏิบัติการ จากนั้นดึงบอทชื่อ @GetChatID_Bot เข้ากลุ่มเพื่อแจ้ง ID และคัดลอก ID (ติดลบ) มาใช้งาน
